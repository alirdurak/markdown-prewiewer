 # Başlık

  Bu bir **Markdown** belgesidir. Markdown, kolayca okunabilen ve yazılabilen basit bir metin biçimlendirme dilidir.
  
  ## Alt Başlık
  
  ### Alt Alt Başlık
  
  Bu belgede, Markdown formatındaki metinleri render etmek için kullanabileceğiniz bir JavaScript kütüphanesi olan `react-markdown` hakkında bilgi verilecektir.
  
  - Markdown formatında metinler, kolayca okunabilen bir biçimde yazılabilir.
  - Markdown, özellikle web siteleri için idealdir çünkü HTML'e dönüştürülebilir.
  - `react-markdown` kütüphanesi, Markdown metinlerini doğrudan React bileşenlerine dönüştürür.
  
  [GitHub](https://github.com/), [Wikipedia](https://www.wikipedia.org/) gibi siteler Markdown formatını kullanırlar.
  
  `Kod örneği`
  
  ```jsx
  import ReactMarkdown from 'react-markdown';
  
  function MyMarkdownComponent({ markdownText }) {
    return (
      <div className="markdown-container">
        <ReactMarkdown>{markdownText}</ReactMarkdown>
      </div>
    );
  }
